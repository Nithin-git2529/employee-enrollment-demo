# Employee Enrollment (Demo)

A minimal demo for enrolling employees:
- Create employee (name, email, department, start_date)
- List all employees
- Update status (ENROLLED/INACTIVE)

> Educational sample. Uses Flask + SQLite + simple HTML/JS.
